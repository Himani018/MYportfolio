# 🌐 Personal Portfolio

A modern, responsive portfolio website built using **React**, **HTML**, **CSS**, and **JavaScript** to showcase my projects, technical skills, certifications, and learning journey as a **Bachelor of Computer Applications (BCA)** student.

## 📖 About

This portfolio serves as my personal website where I showcase:

* 👨‍💻 About Me
* 🛠️ Technical Skills
* 📂 Featured Projects
* 🎓 Education
* 📜 Certifications
* 📈 GitHub & LeetCode Highlights
* 📬 Contact Information

The goal of this project is to maintain a professional online presence while documenting my growth as an aspiring Software Developer and AI enthusiast.
